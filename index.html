Key Features

TypeR enhances the original TyperTools extension with bug fixes and essential new features.

🔄

Bug Fixes

Multiple bugs from the original TyperTools have been fixed for a more stable and reliable experience.

🖋️

Improved Text Smoothing

Fixed the bug that changed text smoothing from "Strong" to "Smooth" when increasing text size.

↔️

Stable Auto-Centering

Text shape no longer changes when using auto-centering.

⌨️

Customizable Shortcuts

Modify keyboard shortcuts to suit your preferences and enjoy the new shortcuts that have been added.

🔍

Automatic Page Detection

Automatically detects pages when importing for a smoother process.

📄

Automatic Page Switching

Automatically switches pages for a seamless workflow.

📏

Resize TypeR

Decreased size limit of the TypeR window for a smaller footprint.

↕️

Line Spacing Sync

Line spacing automatically adjusts when increasing/decreasing text size.

📐

Adaptive Size

If no fixed text size is defined, the size of the selected layer will be used.

Quick Installation

Get started with TypeR in one minute.

1

Extract & Install

Extract the archive and execute the installation script for your operating system.

For macOS, open the terminal, go the TypeR folder and write:

chmod +x install_mac.sh
./install_mac.sh

For Windows execute:

install_win.cmd
2

Finish Setup in Photoshop

That's it! Now go to Window > Extensions > TypeR in Photoshop.

Note for macOS ARM (M1/M2):
You might have to right-click on your Photoshop .app, choose Get Info, and check Open with Rosetta.
